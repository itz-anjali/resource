The Problem

We noticed a big problem. When new students join the college or when exams are near, they struggle to find the right study materials. Books, videos, syllabi, and old question papers are scattered everywhere online, making it stressful and time-consuming to find them.

This problem affects everyone, especially students in the BCA department, 2nd year. We felt it was time for a better way.

Our Solution

Introducing The Resource Hub, a central place for all academic resources.

Centralized Access

No more searching on different websites. All resources for all courses are in one, organized place.

Structured Format

Resources are carefully organized by department, year, subject, and type, making it easy to find what you need quickly.

Interactive & Easy to Use

The website is built to be simple and easy to use on any device, with a clean design and smooth animations.

Technologies Used

The Resource Hub is built using a modern and robust web development stack.

HTML

HTML (HyperText Markup Language) is the foundation of our website. It structures the content and tells the browser what each element is, like headings, paragraphs, and images.

CSS

CSS (Cascading Style Sheets) makes our website beautiful. We used it to add colors, fonts, layout, and the smooth animations you see. We used Tailwind CSS for a modern design.

JavaScript

JavaScript adds the smarts and interactivity. It's used for the smooth scrolling, fade-in animations, and other dynamic features that make the report feel alive.
